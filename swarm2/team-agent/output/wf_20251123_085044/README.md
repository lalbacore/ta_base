# Generated Application

## Mission
Generate a comprehensive 20-page medical reference guide on hormone replacement therapy for women, suitable for healthcare professionals

## Architecture
- Type: generator_pattern
- Components: content_generator, section_builder, formatter, publisher

## Tech Stack
- Language: Python 3.9+
- Testing: pytest

## Usage
Run the application:
```bash
./run.sh
```

## Testing
Run tests:
```bash
pytest
```
